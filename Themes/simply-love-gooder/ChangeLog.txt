ChangeLog (Version 7 - "Fixes and Fixes")

- Fixed ColorRGB Issue
	This took a long time to figure out.
	Now the theme can function corretly while switching via the theme switcher!

- Foreign character support
	This feature is a recreation of dbk2's jfonts workaround.
	Now the theme can display many japanese and korean characters without any issues.

- Fixed issues with Wineskin
	Wineskin had some problems trying to run the theme. The theme should now be stable with the game running with new scripts.
	HOWEVER, If you're using Wineskin, don't use 'Song Backgrounds on Select Music' as it creates bad sectors!

- Could not restore Marathon mode
	I'm sorry... But marathon mode is more broken that i initially thought it was. I don't think that many people use Course Mode anymore in NITG V2.
	I know that there are people who still want to use it. But please hang on until the future of SLGJUVM.
